# MarkSyncr

Sync browser bookmarks across Chrome, Firefox, Safari, Edge, Brave, and Opera.

## What you can do

- Sync bookmarks two-way across multiple browsers automatically
- Store bookmarks in your own GitHub repo, Dropbox, Google Drive, or local file
- Use MarkSyncr Cloud for zero-setup managed sync
- Share bookmark folders with your team (Team plan)
- View and restore bookmark version history
- Import/export bookmarks as HTML or JSON

## Plans

- **Free** — Personal use, MarkSyncr Cloud or BYOS, Chrome/Firefox/Edge
- **Pro** ($5/mo) — Adds Safari, priority sync, 30-day history
- **Team** ($12/mo) — Adds shared folders, team management, 1-year history
